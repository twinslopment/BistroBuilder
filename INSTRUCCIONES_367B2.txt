BISTRO BUILDER 367B2 — HOTFIX DE VISIBILIDAD PARA PRUEBAS DE EDITOR

CAUSA
- Los métodos TryTransitionLine y TryCancel pertenecen al agregado runtime y
  están declarados como internal.
- Las pruebas se compilan en Assembly-CSharp-Editor, mientras el runtime se
  compila en Assembly-CSharp.
- Faltaba declarar el ensamblado de Editor como ensamblado amigo.

CORRECCIÓN
- Añade InternalsVisibleTo("Assembly-CSharp-Editor").
- Mantiene los mutadores del agregado como internal; no se abre una API pública
  que permita modificar comandas fuera del servicio de aplicación.
- Incluye también las correcciones CS0177 de 367B1, por lo que es acumulativo.
